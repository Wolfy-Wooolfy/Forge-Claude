# Project Vision: nextjs_tasks_demo

## Source Summary

This project is a minimal task tracker built with Next.js 14, featuring a frontend that displays tasks and a backend API for managing them. Tasks are stored in-memory, meaning they are lost upon server restart. The application is implemented entirely in TypeScript and does not use any external state management libraries.
